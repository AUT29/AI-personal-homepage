### 国家电网规章制度智能问答系统
#### 项目概述：
面向国家电网规章制度文档，搭建“文档拆分—问答数据集构建—大模型微调—检索增强生成（RAG）”的问答系统，实现领域知识智能问答。
#### 工作内容：
- 数据集构建：将原始 Word 文档拆分为若干Chunk，并基于正则规则从中抽取条款和规定；调用LLM API（DeepSeek-V3）依据条款和规定的内容自动生成问答对，得到 JSON 格式的问答数据集。
- 大模型微调：基于问答数据集，在 Llama-Factory 中对 Llama-8B 进行指令微调，完成训练配置、数据对接与问答测试。
- RAG 增强：引入检索增强生成（RAG）技术，使用 BGE 文本嵌入模型对文档Chunk进行向量化并基于 FAISS 建立索引，依据向量相似度检索与用户提问相关的文本片段，最终由LLM生成回答。
- 系统实现：基于 Flask + SocketIO 搭建流式问答服务，后端将 RAG 检索结果与用户问题拼接后送入LLM进行推理，并将生成内容分片实时回传前端，实现低延迟的实时对话体验。

### 基于大模型与多跳推理的智能问答方法研究
#### 项目概述：
研究基于大模型多跳推理、多智能体协作与动态路由的智能问答方法
#### 工作内容：
- 基于通义灵码、Cursor、CodeX 等AI编程工具辅助完成核心代码的编程、调试与迭代；设计对比和消融实验，分析实验数据，撰写学术论文。
- 知识图谱会话推理：针对知识图谱对话式问答场景，基于 PyTorch 框架实现知识图谱推理流程：图神经网络（GNN）检索并裁剪候选关系 -> LLM生成“用户意图+推理指导” -> PPO 强化学习训练路径选择策略。
- 命题网络检索：针对开放域问答场景，首先将文档拆分为原子命题（实体解析+共指消解+复句拆分），并基于命题间的实体对齐构建命题网络；根据用户问题，采用 Beam Search 进行命题组合检索，提升多跳证据的召回质量。
- 多智能体RAG：基于高层LLM对用户问题进行依赖感知的子查询分解，动态分配 BM25/Dense/PNet/Direct 智能体（低层大模型）；多轮执行“子查询检索 -> 低层信息摘要 -> 高层信息汇总 -> 子查询重构/生成回答”的步骤，低层LLM智能体支持子查询重写、短期记忆、智能体切换。

### 面向智能终端视频设备的区域人员密度检测 
#### 项目概述：
基于树莓派和YOLO目标检测方法，实现在智能终端视频设备上对区域人群密度进行实时监测
#### 工作内容：
- 基于 PyTorch 深度学习框架对 YOLOv5 进行轻量化改进，引入 FasterNet 轻量级主干网络，并在训练阶段采用 OTA 标签分配策略，提高模型推理速度的同时提升高密度场景下行人目标的检测精度。
- 实现 YOLOv5 + CSRNet 的组合检测方案：低密度场景采用目标检测计数，高密度/遮挡场景采用密度图回归，兼顾复杂场景下计数的稳定性和效率。
- 负责技术方案的落地，搭建基于树莓派的边缘计算原型，完成视频流采集、模型推理与结果可视化全流程，输出人群密度热力图与人数估计结果，支持实时监测与人群过度密集时告警。
